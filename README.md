# Quiz-App-React
A React quiz app to test your knowledge. Built with **React** and **Tailwind CSS**.

## Live Demo
https://quizreactx.netlify.app/

## Features

* Multiple choice questions
* Tracks user score
* Next and Previous navigation
* Highlights question and options
* Responsive design


## Tech

* React
* Tailwind CSS
* JavaScript (ES6+)

## Setup
git clone https://github.com/yourusername/react-quiz-app.git
cd react-quiz-app
npm install
npm run dev


## How It Works

* `App.jsx` manages questions, current question index, and score
* `QuesBox` shows the question and options, updates score on click
* Final score displays after last question

## Customize

Edit the `questions` array in `App.jsx` to add or change questions.
